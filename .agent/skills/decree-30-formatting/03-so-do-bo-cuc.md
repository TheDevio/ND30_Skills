## 2. Chi tiết vị trí từng thành phần

### Góc trên bên TRÁI (≈ 1/2 trang trái)

| Thành phần | Vị trí | Căn lề |
|-----------|--------|--------|
| **(1)** Tên CQ chủ quản | Hàng đầu, bên trái | **Giữa** (trong nửa trái) |
| **(1)** Tên CQ ban hành | Dưới CQ chủ quản | **Giữa** (trong nửa trái) |
| Dòng kẻ | Dưới tên CQ ban hành | Giữa, ngắn |
| **(3)** Số, ký hiệu | Dưới dòng kẻ | **Giữa** (trong nửa trái) |

### Góc trên bên PHẢI (≈ 1/2 trang phải)

| Thành phần | Vị trí | Căn lề |
|-----------|--------|--------|
| **(2)** Quốc hiệu | Hàng đầu, bên phải | **Giữa** (trong nửa phải) |
| **(2)** Tiêu ngữ | Dưới Quốc hiệu | **Giữa** (trong nửa phải) |
| Dòng kẻ | Dưới Tiêu ngữ | Giữa, dài bằng dòng chữ |
| **(4)** Địa danh, thời gian | Dưới dòng kẻ | **Giữa** (trong nửa phải) |

### Giữa trang (toàn bộ chiều ngang)

| Thành phần | Vị trí | Căn lề |
|-----------|--------|--------|
| **(5a)** Tên loại VB | Dưới phần header | **Giữa** |
| **(5b)** Trích yếu | Dưới tên loại VB | **Giữa** |
| Dòng kẻ | Dưới trích yếu | Giữa |
| **(6)** Nội dung VB | Phần chính | **Đều hai bên** (Justified) |

### Góc dưới bên PHẢI

| Thành phần | Vị trí | Căn lề |
|-----------|--------|--------|
| **(7a)** Quyền hạn, chức vụ | Dưới nội dung, bên phải | **Giữa** (trong nửa phải) |
| **(7b)** Chữ ký + Họ tên | Dưới chức vụ | **Giữa** (trong nửa phải) |

### Góc dưới bên TRÁI

| Thành phần | Vị trí | Căn lề |
|-----------|--------|--------|
| **(8)** Nơi nhận | Dưới nội dung, bên trái | **Trái** |
| **(11)** Ký hiệu người soạn | Góc dưới cùng bên trái | **Trái** |

## 3. Sơ đồ bố cục — Công văn

Công văn có **khác biệt** so với văn bản có tên loại:

```
┌────────────────────────┬──────────────────────────────────┐
│  TÊN CQ CHỦ QUẢN       │  CỘNG HÒA XÃ HỘI CHỦ NGHĨA    │
│  TÊN CQ BAN HÀNH       │       VIỆT NAM                   │
│  ────────               │  Độc lập - Tự do - Hạnh phúc   │
│                         │  ──────────────────              │
│  Số:     /...-...       │  ..., ngày ... tháng ... năm ... │
│  V/v .................. │                                   │ ← Trích yếu ở đây
├─────────────────────────┴──────────────────────────────────┤
│                                                             │
│         Kính gửi: ...............................           │ ← Căn giữa hoặc trái
│                                                             │
│    (Nội dung công văn)                                      │
│    ...................................................../. │
│                                                             │
├──────────────────────┬─────────────────────────────────────┤
│  Nơi nhận:           │         QUYỀN HẠN, CHỨC VỤ         │
│  - Như trên;         │         (Chữ ký, đóng dấu)          │
│  - ...;              │                                      │
│  - Lưu: VT, ...     │         Họ và tên                   │
└──────────────────────┴─────────────────────────────────────┘
```

> **Lưu ý**: Công văn **KHÔNG** có tên loại văn bản (QUYẾT ĐỊNH, CHỈ THỊ...).
> Trích yếu đặt ở **dưới Số, ký hiệu** (bên trái), không đặt ở giữa trang.

## 4. Khoảng cách giữa các thành phần

| Từ → Đến | Khoảng cách |
|----------|-------------|
| Tên CQ chủ quản → Tên CQ ban hành | **0** dòng trống (liền kề) |
| Tên CQ ban hành → Dòng kẻ | **0** |
| Dòng kẻ → Số, ký hiệu | Khoảng **6 pt** trước |
| Quốc hiệu → Tiêu ngữ | **0** dòng trống |
| Tiêu ngữ → Dòng kẻ | **0** |
| Dòng kẻ → Địa danh, thời gian | Khoảng **6 pt** trước |
| Số, ký hiệu → Tên loại VB | **1–2** dòng trống |
| Tên loại VB → Trích yếu | **0** (liền kề) |
| Trích yếu → Dòng kẻ → Nội dung | **1** dòng trống |
| Nội dung → Chức vụ người ký | **1–2** dòng trống |
| Chức vụ → Chữ ký | Khoảng **3–4** dòng trống (cho chữ ký) |
| Chữ ký → Họ tên | **0** |

---

## 5. Quy tắc chia cột

| Phần | Tỷ lệ chia cột |
|------|----------------|
| Header (CQ + Quốc hiệu) | **≈ 40% trái : 60% phải** |
| Nội dung chính | **100% toàn bộ** |
| Footer (Nơi nhận + Ký tên) | **≈ 50% trái : 50% phải** |
